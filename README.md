## Strategic frame

```
Vibe Marketing  →  category (collective; emerged publicly 2024–2025)
       ↓
MAAS™           →  operating system (Believe Global, since 1998)
       ↓
Believe Global  →  the firm that installs MAAS for LATAM enterprise
       ↓
maasy           →  cloud interface that scales the system post-install
```

**Believe Global does not claim authorship of the Vibe Marketing term.** The category took public shape in 2024–2025 through multiple voices — Andrej Karpathy (vibe coding as matrix), Greg Isenberg, James Dickerson, Jordan Mix (movement formalization), Rob Cressy (VIBE Framework), and others.

What Believe Global claims is **methodology**: MAAS™, the operating system that has been running what we now call Vibe Marketing since 1998. Different domain. Different timeline. Same operational principle.
